# Wadhiya Khaba Website - GitHub Version

Welcome to your clean website project. Use this folder to manage and update your website on GitHub Pages.

## 🚀 How to Update Your Website
1.  **Edit your files**: Make changes to `index.html`, `style.css`, or add new images.
2.  **Sync to GitHub**: Double-click **`push.bat`**. This will:
    *   Add your new files.
    *   Commit them with a message.
    *   Push them to your [GitHub Pages link](https://asoffical50-dot.github.io/wadhiyaakhabaoffical/).
3.  **Wait for 60 seconds**: GitHub will update the live site. (Use **Ctrl + F5** in your browser to see the latest changes).

## 🖼️ Image Optimization
If you add large images (more than 1MB), double-click **`compress.ps1`** (run with PowerShell) to make them smaller. This ensures your website stays fast and stays under the 100GB limit.

## 📁 Folder Structure
*   `index.html`: Main website page.
*   `style.css`: Website design.
*   `script.js`: Interactive features.
*   `extension/`: Files for your browser extension.
*   `push.bat`: Your one-click update tool.
